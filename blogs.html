Local storage and Session storage

Local storage :
1: We can access the data of local storage whenever we want.
2: The data which is added in local storage, that couldn't gonna remove until we remove those data manually.

Session storage :
1: We can access the data of session storage when we are in the tab.
2: The data which is added in the Session storage, that will remove when we close the browser or browser tab.

Global scope and Block scope

Global scope :
It can be accessed from anywhere in JavaScript program and from other scope's

Block scope :
Are accessible within that particular condition or loop.This scope restricts the variable that is declared inside a specific block, from access by the outside of the block. The let & const keyword facilitates the variables to be block scoped.
